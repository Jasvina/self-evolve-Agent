# self-evolve-Agent

本仓库用于沉淀课题 **“产业级记忆 - 自我演化智能体的轨迹分析与长期记忆能力探索”** 的调研、设计与后续实验资产。

## 当前内容

- [调研报告：产业级记忆与自我演化智能体](docs/research/industrial_memory_self_evolving_agents.md)

## 课题定位

面向高难算法任务与长程智能体任务，本课题关注如何从自我演化智能体的状态轨迹、行为选择、评估反馈与修复过程里抽取可复用、可检索、可验证的长期记忆，并把这些记忆用于未来相似任务中的动态优化。

当前建议把“记忆”视为一个可工程化的闭环：

```text
轨迹日志 -> 片段定位 -> 记忆抽取 -> 证据校验 -> 索引/存储 -> 检索注入 -> 任务收益评估 -> 记忆更新
```

## 建议下一步

1. 固化轨迹日志到记忆对象的 JSON schema。
2. 先在受控任务/伐谋已有长轨迹日志上做离线记忆抽取。
3. 用 no-memory、raw-retrieval、structured-memory 三组对照验证记忆是否真正提升任务成功率与恢复效率。
